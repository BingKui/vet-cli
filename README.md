# vet-cli

[VueElectronTemplate](https://github.com/BingKui/VueElectronTemplate)配套使用cli，用于创建项目等。

## 命令

```bash
# 安装
npm install -g vet-cli
# 新建项目
vet new
# 查看版本
vet -v
# 或者
vet --version
# 查看帮助
vet --help
```
